# RetroLogic 8051 Tic-Tac-Toe


A standalone Tic-Tac-Toe arcade console engineered entirely in 8051 Assembly. This project was developed to solve the **4705 Complex Engineering Problem**, demonstrating bare-metal system architecture, strict memory management within a 128-byte RAM limit, and hardware-offloaded UI rendering.

## System Architecture
- Solves the 4705 Complex Engineering Problem.
- O(1) Heuristic Priority AI Engine.
- Hard stack-flush memory management (0% crash rate).
- LM044L LCD hardware offloading.
- 3x3 matrix keypad polling engine.

## Requirements
- **Proteus Design Suite 8.13** (Required to run the `.pdsprj` hardware simulation without compatibility errors).
- Standard 8051 Assembler like MIDE-51 (Only required if recompiling the `.asm` source code).
